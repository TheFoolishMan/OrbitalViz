Orbital Viz


OrbitalViz is a 3D web application for orbital visualization.
This application is the outcome of my college Design Project(CHEM F376) under the supervision of Dr. Prashant U Manohar.
The main aim to develop this project was to create something for Undergraduate(or High School) Chemistry students to help them in visualization of various orbitals. This application uses quantum equations to calculate both hybridized and unhybridized orbitals precisely. Various parameters are available under Advanced Settings to tune the orbital properties.

Main feature of this application one can try mixing of different orbitals with appropriate coefficients to observe how the combination look like. I hope this project will reach its aim to help chemistry students in their understanding of orbitals. If you are a chemistry student/professor and you liked this application, please share it with your friends and colleagues.

Thank You!

Instructions

Movement Instructions
Controls are like standard 3D video games. Use arrow keys or W,A,S,D to move forward, left, backward, right respectively in the area. To look around in the scene, Left-Click and drag.

Creating Orbitals
Click hybridized/unhybridized button to create respective orbitals. Different parameters will appear to enter info about the orbitals.
For unhybridized orbitals,
Enter n,l,m quantum number values to define the orbital where n is principal quantum number, l is orbital angular momentum quantum number and m is magnetic quantum number. Click Visualize button to load the orbital.
For hybridized orbitals,
Enter quantum number values to define the orbital and the normalized coefficient value of the respective orbitals. Click on Add/remove orbitals for adding/removing orbitals. One can see the equation of final hybridized orbital. After entering all values, click Visualize button.

Advanced Settings
Various parametrs are available to customize the orbital according to one's requirement. This can be used if application is taking a lot of time to load the orbitals. One can compensate between speed and accuracy of the orbital formation.
Remember, one has to load the orbital again to apply the changes. Significance of different parameters is discussed as follows:-

Orbital Scale is the size of the orbital spread over the scene. Higher the value, more the size of the orbital.

Color Intensity is the intensity of the each point/line forming the orbital. Change this value for the brightness of the orbital.

Maximum Radius is the radius of the orbital. This is the value which is used in quantum equation to calculate the orbitals. More the radius, more time it will take to load the orbital.(Remember, as we go away from nucleus, probability of finding electron decreases. Therefore, very less points at farther radius.)

Step Size is the number of steps in between two successive integers of the radius value. More the step size, more information in the orbital.

Iterations will give you the estimate of how many iterations it will take to load the orbital. Changing other values will change this value to give you the measure how much time will it take to load the orbital.

Mesh Topology will decide to make the orbitals with line or points. On web application, only Line topology is there due to limitation in Unity software to render points for WebGl Build. On Windows version, both topologues are available.

Line Length is the length of the line in Line Mesh Topology. Adjust it if tour model doesn't look good.

RGB Sliders are available to change the color of of the orbital for its positive and negative part.
